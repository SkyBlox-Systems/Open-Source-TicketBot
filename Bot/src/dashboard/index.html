How To use

To use our transcript, you should of got a link posted in your transcript channel saying 
The transcript for ticket: ticket-ID. The file can be found below or by our server: "shard1.ticketbots.tk/SERVERID/GeneratedID.html" Enter that into your browser.  It should redirect you to a website page with the transcript. If it gives an error, it might be because you have entered the URL wrong or the transcript is not created. Make sure that you have it enabled on your server.
